Chrome One Tap Login for WooCommerce allows the customers to log in automatically into a website without having to enter passwords each time a customer visits a store

== Feature of Plugin ==

1. Allows the customers to login into a store, automatically.
2. A smooth process as the customers will not have to remember passwords everytime they log in.
3. Multiple account credentials can be saved by customers.
4. Customers can view and delete the saved passwords.
